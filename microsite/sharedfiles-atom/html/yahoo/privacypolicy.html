About cookies.

Summary: A cookie is a small text file containing an identifier (a string of numbers and letters) that is sent by a web server to a web browser and is stored by the web browser. The identifier is then sent back to the web server every time the web browser appeals to a page from the server.

Cookies can be either "persistent" cookies or "session" cookies:

  1. A persistent cookie will be stored by a web browser and will remain justifiable until its set expiration date, unless removed by the user before the expiration date
  2. A session cookie will expire at the end of the user’s session, i.e. when the web browser is closed.
